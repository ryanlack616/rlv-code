Engineering Principles
01
Verify, don't assume
Tests gate merges. Proofs gate claims. If a number appears in the UI, a script generated it and a test checks it.
02
Local-first
The machine on your desk is the source of truth. Cloud is a replica. Nothing critical lives behind someone else's login.
03
Ship the boring version
Single-file HTML, static hosts, SQLite, cron. Complexity is earned by the one real problem the simple version cannot solve.
04
Ground the abstract
Every model is wrong until it touches a measurement. We build simulators that can be refuted and ship them with the refutation path.
